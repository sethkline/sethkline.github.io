Selected Projects

Reverence Studios Website

Tech Stack: Nuxt 3 (Vue), Tailwind, Storyblok (Headless CMS).

Problem: The old site was built on WordPress, then migrated to Gatsby, but it had no database and was difficult to update (needed PRs). The studio team wanted a simpler, more maintainable way to add or edit content.

Solution: I rebuilt the site using Nuxt (Vue framework) integrated with the Storyblok headless CMS. I created reusable Vue components styled with Tailwind and Headless UI, allowing content editors to manage classes, schedules, and announcements via a user-friendly CMS interface—no more code pushes required.

Result: The new site is much easier to maintain. Non-technical staff can update classes and announcements in real-time. The new UI is fully mobile-friendly, using a color palette aligned with the studio’s brand.

Online Ticket Sales for Reverence Studios

Tech Stack: Nuxt 3 (Vue), PrimeVue, Strapi (Node.js), Stripe.

Problem: The old ticket system was a custom PHP app that was difficult to maintain and lacked modern features like seat selection and easy payment processing.

Solution: I developed a new online ticket platform where users can choose seats for multiple shows and securely pay via Stripe. The front end uses Nuxt 3 and PrimeVue for a dynamic yet consistent UI, while the back end runs on Strapi for content/data management.

Result: The studio can now easily manage event details and track ticket sales from a central admin panel. Customers get a smoother, modern e-commerce-like experience with seat maps and reliable checkout flow.
